Вот **современный стиль дизайна для портфолио-лендинга в тренде 2026 года**. Такой дизайн используют стартапы и топ-разработчики. Он выглядит **минималистично, технологично и дорого**. 🚀

---

# Общая концепция дизайна

**Стиль:**
Tech Minimal / Futuristic UI

**Основные идеи:**

* минимализм
* большие заголовки
* много воздуха
* плавные анимации
* стеклянные элементы
* мягкие градиенты
* тёмная тема

---

# 1. Цветовая палитра

Лучше всего делать **тёмную тему**.

### Основной фон

```
#0F0F14
```

или

```
#0B0B0F
```

---

### Акцентный цвет

Градиент:

```
#6C5CE7 → #00D4FF
```

или

```
#7C3AED → #22D3EE
```

---

### Дополнительные цвета

текст

```
#E4E4E7
```

вторичный текст

```
#A1A1AA
```

карточки

```
#1A1A22
```

Макет:

```
------------------------------------------------

Большой заголовок

Короткое описание

[ Кнопка ]   [ Кнопка ]

Иллюстрация / 3D / анимация
------------------------------------------------
```

### Пример текста

Заголовок

```
Frontend Developer
Building modern web experiences
```

Описание

```
I create fast, responsive and beautiful websites
for startups and businesses.
```

---

# 4. Карточки (glassmorphism)

Тренд 2026 — **стеклянные карточки**.

CSS идея:

```
background: rgba(255,255,255,0.05);
backdrop-filter: blur(10px);
border: 1px solid rgba(255,255,255,0.1);
border-radius: 16px;
```

Карточки используются для:

* проектов
* навыков
* услуг

---

# 5. Кнопки

Кнопки должны быть **градиентные**.

Пример:

```
background: linear-gradient(
90deg,
#7C3AED,
#22D3EE
);
```

Hover эффект:

* свечение
* увеличение

---

# 6. Анимации (очень модно)

Добавь:

### hover анимации

карточка немного поднимается

```
transform: translateY(-5px);
```

---

### scroll анимации

появление элементов

```
fade in
slide up
```

---

### typing animation

текст печатается:

```
I build websites
I design interfaces
I create web apps
```

---

# 7. Блок проектов

Сетка:

```
2 или 3 карточки в ряд
```

Карточка:

```
[ изображение проекта ]

Название

описание

технологии

[ Watch Demo ] [ GitHub ]
```

Hover эффект:

* картинка увеличивается
* появляется кнопка

# 8. Навыки

Лучше сделать **иконки технологий**.

Пример:

```

CSS
JavaScript
React
Next.js
Node.js
Figma
```

Можно добавить **progress animation**.

Тренд 2026 — **мягкий градиент + шум**.

Пример:

```
radial gradient
```

или

```
blur shapes
```

например:

* фиолетовое пятно
* голубое пятно
